## 开发目标（短期）

### 解决 mysql 配置问题

不同的环境安装方法稍有不同，节省大家配置，并且方便大家统一同时管理数据数据库表。引入 docker.期望所有关于 MySQL 的配置能自动化完成

1. [docker 安装](http://www.runoob.com/docker/ubuntu-docker-install.html)

2. docker 改源

3. 下载 mysql 镜像

```
docker pull mysql(开发使用5.7)
```

4.  制作镜像

    - 自动创建表

    * **数据的备份和恢复 暂时做不到，关机数据库数据可能会丢失**

```
    docker image build -t code_exam_db .
```

5.  启动 mysql 容器

```
    docker run -d -p3306:3306 -e MYSQL_ROOT_PASSWORD=123qweasd --name=db1 code_exam_db
```
